Android逆向学习—环境配置以及工具下载安装

开启USB调试（自己买的Android测试机）

买一台真机，测试机，设置 -> 开发者选项 -> 启用 USB 调试

ADB（Android Debug Bridge，Android 调试桥 ）工具安装

关于ADB（Android Debug Bridge，Android 调试桥 ）介绍：https://developer.android.com/studio/command-line/adb

下载：https://developer.android.com/studio/releases/platform-tools#downloads ，点击下载” 下载适用于 Windows 的 SDK Platform-Tools”，解压之后会有adb.exe

MT管理器

MT 管理器是一款文件管理工具和 APK 逆向修改神器。软件有汉化应用、替换资源、修改布局、修改逻辑代码、资源混淆、去除签名校验等功能

百度自行安装即可

模拟器

雷电模拟器：https://www.ldmnq.com/

JDK

华为镜像源下载：https://mirrors.huaweicloud.com/java/jdk/

配置环境变量，输入”java -version”显示结果即可

SDK

下载地址：https://www.androiddevtools.cn/

配置环境变量，输入”ddms”检查是否安装成功

NDK

官方介绍

下载地址：https://developer.android.com/ndk/downloads?hl=zh-cn

配置环境变量，输入”ndk-build”检查是否安装成功
